# Golden Poopers - Airbean API
Detta är ett RESTful API byggt i Node.js med Express och MongoDB, som möjliggör att användare kan skapa konton, se kaffemenyn, lägga produkter i kundvagnen och beställa kaffe.

---

## Projektbeskrivning

Airbean API är ett grupprojekt där målet är att skapa ett komplett backend-API enligt en given dokumentation. API:et simulerar en kaffebar där användare kan:
- Se kaffemenyn
- Skapa konto och logga in
- Lägga till/ta bort produkter i en kundvagn
- Lägga en order
- Se tidigare ordrar kopplade till ett unikt användar-ID

**Notera:** Detta projekt innehåller endast backend. Ingen frontend ingår.
